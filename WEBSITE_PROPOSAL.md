# WISE Institute Website Proposal

## Overview
A simple, clean website focused on sharing course information for WISE Institute. Minimal maintenance required (2-3 updates per year).

## Reference
- **Inspiration**: https://bcdentalstudyclub.ca
- **Content Scale**: 1/10 of reference site (much smaller, focused)
- **Update Frequency**: 2-3 times per year (when new course dates are announced)

## Website Structure

### Pages (3-4 pages total)

#### 1. **Home Page** (`/`)
**Purpose**: Quick overview and course highlights

**Content**:
- Hero section with main course image
- Brief welcome message (2-3 sentences)
- Upcoming courses preview (next 2-3 courses with dates)
- Quick links to:
  - View all courses
  - Contact us
  - About WISE Institute

**Design**: Clean, minimal, professional

---

#### 2. **Courses Page** (`/courses`) 
**Purpose**: Complete course listing and details

**Content**:
- Course list with:
  - Course name
  - Dates (year, months)
  - Duration
  - Brief description (2-3 lines)
  - Registration status (Open/Closed/Upcoming)
  - Link to contact/register
- Past courses (optional, collapsed section)
- Course format information (if applicable)

**Design**: Simple table or card layout, easy to scan

---

#### 3. **About Page** (`/about`)
**Purpose**: Information about WISE Institute and directors

**Content**:
- Brief about WISE Institute (1-2 paragraphs)
- Directors section:
  - Dr. Chris Lee (photo, brief bio)
  - Dr. Stephen Yoon (photo, brief bio)
- Mission/vision (optional, 1 paragraph)

**Design**: Clean, professional, photo + text layout

---

#### 4. **Contact Page** (`/contact`)
**Purpose**: How to reach WISE Institute

**Content**:
- Contact information:
  - Email
  - Phone (if applicable)
  - Address (if applicable)
- Contact form (optional, simple)
- Office hours (if applicable)

**Design**: Simple, accessible contact information

---

## Design Principles

### Simplicity
- Clean, uncluttered design
- Focus on course information
- Easy to navigate
- Fast loading

### Mobile Responsive
- Works perfectly on all devices
- Touch-friendly navigation
- Readable on small screens

### Maintenance-Friendly
- Easy to update course dates
- Simple content management
- No complex features that break

### Professional
- Reflects the quality of WISE Institute
- Trustworthy and credible
- Modern but not flashy

## Content Updates

### When to Update (2-3 times per year)
1. **New Year**: Update course dates for the upcoming year
2. **Mid-Year**: Add any new courses or update dates
3. **As Needed**: Update contact info, director bios, etc.

### What to Update
- Course dates on Courses page
- Course descriptions (if changed)
- Contact information (if changed)
- Director information (if changed)

### Update Process
- Simple text/content edits
- No technical knowledge required
- Can be done in 15-30 minutes

## Maintenance Guide

### How to Update Course Dates
1. Go to `/courses` page
2. Find the course
3. Update the date
4. Save

### How to Add New Course
1. Go to `/courses` page
2. Add new course entry
3. Fill in details (name, date, description)
4. Save

### Estimated Time
- Update course dates: 5-10 minutes
- Add new course: 10-15 minutes
- Update contact info: 2-5 minutes
