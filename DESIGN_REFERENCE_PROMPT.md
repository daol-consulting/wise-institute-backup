# Directors Page Design Reference Request Prompt

## 프롬프트 (한국어)

```
의료진 소개 페이지 디자인 레퍼런스를 제안해주세요.

**프로젝트 컨텍스트:**
- 웹사이트: WISE Institute (치과 임플란트 교육 기관)
- 페이지: Directors (의료진 소개 페이지)
- 목적: 두 명의 의사(Dr. Chris Lee, Dr. Stephen Yoon)의 프로필을 깔끔하고 전문적으로 소개

**현재 디자인 포인트 색상:**
- Primary Color (메인 포인트): Teal 계열
  - Primary 600: #0d9488 (주요 액센트)
  - Primary 500: #14b8a6 (보조 액센트)
  - Primary 100: #ccfbf1 (배경/아이콘 배경)
- Secondary Color: Slate/Gray 계열
  - Secondary 900: #0f172a (제목 텍스트)
  - Secondary 700: #334155 (본문 텍스트)
  - Secondary 100: #f1f5f9 (배경)
- Accent Color: Purple 계열 (선택적 사용)
  - Accent 600: #cc2bff

**현재 레이아웃 구조:**
- 각 의사마다 2열 그리드 레이아웃
- 왼쪽/오른쪽: 텍스트 정보 (이름, 학위, 경력, 학회)
- 오른쪽/왼쪽: 프로필 사진 (520x572px)
- 흰색 배경, 깔끔한 미니멀 디자인
- Briefcase 아이콘과 Award 아이콘을 사용한 섹션 구분

**요청사항:**
1. 의료진 소개 페이지에 적합한 현대적이고 전문적인 디자인 레퍼런스
2. 위 색상 팔레트를 활용한 디자인 예시
3. 깔끔하고 미니멀한 스타일
4. 의료/교육 분야에 적합한 신뢰감 있는 디자인
5. 모바일 반응형 레이아웃 고려
6. 사진과 텍스트의 균형잡힌 배치
7. 경력 및 학회 정보를 명확하게 표시할 수 있는 레이아웃

**참고 스타일:**
- 클린하고 모던한 의료진 소개 페이지
- 대학/병원 웹사이트의 교수진 소개 페이지
- 전문 서비스 제공자의 팀 소개 페이지
- 미니멀하면서도 정보 전달이 명확한 디자인
```

## Prompt (English)

```
Please suggest design references for a medical staff introduction page.

**Project Context:**
- Website: WISE Institute (Dental Implant Education Institution)
- Page: Directors (Medical Staff Introduction Page)
- Purpose: Cleanly and professionally introduce two doctors (Dr. Chris Lee, Dr. Stephen Yoon)

**Current Design Point Colors:**
- Primary Color (Main Point): Teal tones
  - Primary 600: #0d9488 (main accent)
  - Primary 500: #14b8a6 (secondary accent)
  - Primary 100: #ccfbf1 (background/icon background)
- Secondary Color: Slate/Gray tones
  - Secondary 900: #0f172a (heading text)
  - Secondary 700: #334155 (body text)
  - Secondary 100: #f1f5f9 (background)
- Accent Color: Purple tones (optional use)
  - Accent 600: #cc2bff

**Current Layout Structure:**
- 2-column grid layout for each doctor
- Left/Right: Text information (name, credentials, career, societies)
- Right/Left: Profile photo (520x572px)
- White background, clean minimal design
- Section separation using Briefcase and Award icons

**Requirements:**
1. Modern and professional design references suitable for medical staff introduction pages
2. Design examples utilizing the above color palette
3. Clean and minimal style
4. Trustworthy design suitable for medical/education field
5. Mobile responsive layout consideration
6. Balanced placement of photos and text
7. Layout that clearly displays career and society information

**Reference Styles:**
- Clean and modern medical staff introduction pages
- Faculty introduction pages from university/hospital websites
- Team introduction pages from professional service providers
- Minimal design with clear information delivery
```

### 디자인 톤 & 매너
- **톤**: 전문적, 신뢰감, 모던, 미니멀
- **매너**: 깔끔한 타이포그래피, 넓은 여백, 명확한 정보 계층 구조
